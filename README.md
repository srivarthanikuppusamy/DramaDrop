# DramaDrop 🎬

DramaDrop is an AI-powered entertainment recommendation chatbot focused on helping users discover:

- 🍿 Movies
- 🍥 Anime
- 📺 K-Dramas
- 🎞 TV Shows
- 💎 Hidden Gems

DramaDrop provides personalized recommendations based on:
- mood
- genre
- emotional vibe
- pacing
- storytelling style
- OTT platform preference
